Agegnehu Global University (AGU)
Welcome to the official repository for Agegnehu Global University (AGU). AGU is a modern digital institution dedicated to academic excellence, innovation, and preparing the next generation of leaders in technology and finance.
🚀 Overview
AGU provides advanced digital education focused on practical skills and industry-relevant curriculum. We leverage modern web technologies and cloud infrastructure to deliver a seamless learning experience.
📚 Academic Programs
We offer specialized programs designed to meet the demands of the digital economy:
Computing & AI: Full-stack development, Software Engineering, and Blockchain architecture.
Business & Finance: Entrepreneurship, Fintech, and Digital Economic systems.
Engineering: Sustainable technology and infrastructure development.
🛠 Tech Stack
This project is built using:
Frontend: HTML5, CSS3.
Backend/Database: Supabase (PostgreSQL) for secure data management.
Hosting: GitHub Pages for scalable web delivery.
🤝 Admissions
Prospective students can apply through our integrated admission portal. All applications are securely stored and processed via our Supabase-backed infrastructure.
